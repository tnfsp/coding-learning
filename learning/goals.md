# 學習目標

## 核心目標

成為能與 AI 有效協作的「心臟外科醫師型」使用者

> 心臟外科醫師不需要會所有醫學知識，但需要：
> - **精準判斷**：知道該切哪裡、不該切哪裡
> - **風險意識**：知道什麼操作會出問題
> - **整體理解**：理解系統如何運作，才能做正確決策

對應到 AI Coding：
- 知道該「要求」什麼
- 知道什麼是「危險」的
- 理解程式「如何運作」的基本邏輯

---

## 附加目標

1. **未來能教學**：把學到的東西整理成可分享的內容
2. **做更多專案**：有信心挑戰更複雜的專案

---

## 里程碑

### Milestone 1：理解現有專案
- [ ] 能解釋自己 Flask app 的每一個主要部分
- [ ] 能解釋自己 Next.js 網站的組件結構
- [ ] 知道環境變數如何保護 API key

### Milestone 2：有效協作
- [ ] 能寫出更精準的需求描述
- [ ] 能快速讀懂 AI 產生的程式碼重點
- [ ] 能自己解讀基本的錯誤訊息

### Milestone 3：安全意識
- [ ] 能識別常見的安全風險
- [ ] 養成「這樣安全嗎？」的習慣
- [ ] 知道如何正確處理敏感資料

### Milestone 4：工具熟練
- [ ] 不害怕終端機
- [ ] 理解 Git 的基本操作和用途
- [ ] 知道套件管理在做什麼

---

## 學習原則

1. **不求精通，但求理解**：目標是理解「為什麼」，不是背誦語法
2. **從自己的專案出發**：用已完成的專案當學習素材
3. **問題導向**：遇到問題時深入學習，而非從頭學起
4. **持續累積**：每次學習都做 Anki 卡片，長期記憶
