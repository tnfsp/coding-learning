## 快速問答模式

用戶有一個具體問題，需要快速得到答案。

### 特點
- 直接回答，不需要完整教學流程
- 不自動存檔（除非用戶要求）
- 簡潔但準確

### 回答格式

1. **直接回答**：先給答案
2. **簡短解釋**：為什麼是這樣
3. **實例**（可選）：如果相關，從學習者專案抽取例子
4. **延伸**（可選）：提示相關的學習主題

### 範例

**問題**：`/quick 什麼是 async/await`

**回答**：
> async/await 是 JavaScript 處理「非同步操作」的語法糖。
>
> **為什麼需要**：程式需要等待某些事情（如 API 回應），但不想讓整個程式卡住。
>
> **在你的專案中**：你的 Next.js 網站在取資料時就用到這個概念。
>
> 想深入了解可以用 `/teach async` 系統學習。
